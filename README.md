# SaaS Backend – FlowTask

This repository contains the backend for the **FlowTask** SaaS application. It provides APIs and services to manage tasks, projects, and user workflows.

## Features

- User authentication and authorization
- Project and task management
- RESTful API endpoints for CRUD operations
- Integration-ready for frontend applications
- Scalable architecture for SaaS deployment

## Prerequisites

- Node.js >= 18.x
- npm >= 9.x
- MongoDB or your preferred database
